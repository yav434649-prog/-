# AstrBot 插件：硅基生命（虚拟日程）

本插件为 AstrBot 提供更真实的“虚拟日程”能力：结合天气与地图路程，按时间段生成地点与路程信息，并在会话中注入角色状态。

## 使用指令

- 查看今日日程：`/silicon show`
- 立即重写（可附加要求）：`/silicon renew 你的补充说明`
- 修改每日自动生成时间：`/silicon time HH:MM`
- 高德/POI/路径诊断：`/silicon diag`

## 配置项（后台可视化）

在插件设置的“虚拟日程”分栏配置：
- 高德开放平台 API Key（服务平台选择“Web服务”）
- 默认城市与基地（建议能被地图检索的地点）
- 时间段模板、出行方式、单段通勤上限、天气影响强度
- 创意池与 Prompt 模板

## 依赖

按需安装 `requirements.txt` 中的依赖。

## 开发

主要代码位于：
- `main.py`：插件逻辑、指令、状态注入
- `_conf_schema.json`：可视化配置 Schema
- `metadata.yaml`：插件元数据（名称/描述/作者等）
